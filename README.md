# OpenCV Fitting Demo (AI-assisted)

### 🎯 目标

项目需要对图像进行检测，确保设备能够精确捕捉到图像的参数。

遵循MVP原则，实现一个最小 Demo测试摄像头采集到的图片能否被霍夫转换算法准确识别以及使用最小二乘法是否能够将拟合结果控制在可接受的误差范围内。



### 🛠 技术要点 

- C++ + OpenCV
- 支持随机噪声



### 🔥实现思路

- 先通过灰度、高斯、canny等图像预处理获取轮廓点，再通过图形算法获取几何对象。

- 在设置选项中绑定各个关键参数，实现动态调整参数的效果，以确保能够针对图像的分辨率给出合理的参数。

  

### 🤖 开发方式

使用Trae+Gemini-3进行开发。

代码使用 AI 生成，但数据构造、验证与说明由我独立完成。



### 📝 结果

- 使用霍夫转换具备良好的健壮性，抗噪点效果更强，但参数对图像的限制较大，无法对多种类型的产品进行准确识别。
- 最小二乘法拟合直线会将直线无限延申，若要使用还得加端点限制。
- 最小二乘法抗噪点能力较弱，容易因为干扰点的出现导致拟合的线条误差较大，不符合实际需求。
